### Focus Open Editor
````
ctrl + k -> e
````

---

## Bottom

### Terminal
- show/hide
````
ctrl + j
````
- open new terminal
````
ctl + shift + `
````
- return to editor
````
ctrl + 1
````

---

## Left Sidebar

### Sidebar
- show/hide
````
ctrl + b
````
- focus Explorer
````
ctrl + shift + e or ctrl + 0
````
- focus Open Editors
````
ctrl + k -> e
````

---

## Right Sidebar

### Copilot Chat
- show/hide
````
ctrl + alt + i
````

---

## Editor

### Select
- bulk select
````
ctrl + alt + ↑↓
````
````
// selected all
shift + a
````

### Move
- bulk move
````
ctrl + [ or ]
````

### Focus
- focus Sidebar (Explorer)
````
ctrl + 0
````
- focus Terminal
````
ctrl + `
````
- focus Editor (from Sidebar/Terminal)
````
ctrl + 1
````

### Zen Mode
````
ctrl + k -> z
````

### Markdown Preview
````
ctrl + shift + v
````
